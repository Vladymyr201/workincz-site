# PROJECT_KNOWLEDGE.md

## Архитектура проекта

### Основные компоненты
- **Гибридная платформа поиска работы** - основная система с расширенными ролями пользователей
- **Маркетинговая система** - реферальная программа, email-маркетинг, лендинги, конверсионная аналитика
- **A/B тестирование** - тестирование различных подходов и оптимизация
- **Внешняя аналитика** - интеграция с Google Analytics, Facebook Pixel, LinkedIn Insight
- **Движок оптимизации** - автоматическая оптимизация на основе данных
- **Система тестирования и валидации** - автоматизированные тесты и мониторинг
- **Аналитическая панель** - мониторинг производительности и отслеживание ошибок
- **Система машинного обучения для рекомендаций** - персонализированные рекомендации вакансий
- **Система push-уведомлений и real-time уведомлений** - мгновенные уведомления пользователям
- **Система геймификации** - достижения, бейджи, квесты, очки, награды и социальные элементы
- **Система модерации контента** - автоматическая и ручная модерация, фильтрация спама, система жалоб
- **Система подписок и платежей** - различные планы подписок, премиум функции, интеграция с Stripe
- **Система календаря событий** - планирование собеседований, встреч, дедлайнов, интеграция с внешними календарями
- **Система отзывов и рейтингов** - оценка работодателей, кандидатов, вакансий, верификация отзывов, модерация

### Технологический стек
- **Frontend**: JavaScript ES6+, HTML5, CSS3 (Tailwind CSS)
- **Backend**: Node.js (готов к интеграции)
- **База данных**: Firebase Firestore (готов к интеграции)
- **Аналитика**: Chart.js, собственная система метрик
- **ML/AI**: Алгоритмы рекомендаций, анализ поведения пользователей
- **Уведомления**: Service Worker API, WebSocket, Browser Notifications API
- **Геймификация**: Система достижений, бейджей, квестов, очков
- **Модерация**: Автоматические фильтры, ручная модерация, система жалоб
- **Платежи**: Stripe API, мультивалютность, автоматические платежи
- **Календарь**: iCal формат, интеграция с внешними календарями, система напоминаний
- **Отзывы**: Система верификации, модерация, бейджи, категорийные рейтинги
